This Readme.txt file describes the contents of this folder.

---------
Contents
---------
- STM32F103RB-Nucleo/STM32L053R8-Nucleo/STM32F401RE-Nucleo/STM32L476RG-Nucleo: 
			 this program is supported in platforms: L0, F1, L4 and F4. The binary provided
			 in each platform folder needs to be flashed to the board to use the platform
			 as a flashing platform for the expansion module firmware. The demonstrator GUI
			 is used as the other program running on the PC for this tool. Details are in the
			 step by step guide in the /Documentation folder.
